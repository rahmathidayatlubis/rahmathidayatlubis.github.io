About Me.

Hello! I'm Rahmat Hidayat Lubis, an enthusiastic web developer from Royal University with an academic background in Information Systems. I have a strong passion for designing efficient and elegant digital solutions.

My main focus is on backend development using PHP, with in-depth knowledge of the Laravel and CodeIgniter frameworks. I’m also proficient in using MySQL to design and manage reliable, well-structured databases.

On the frontend side, I have experience using JavaScript to build interactive features, and I’m currently exploring Tailwind CSS to implement modern, responsive styling. Additionally, I have basic knowledge of Java.

Throughout my journey as a student, I have successfully completed various projects, both for academic assignments and freelance work, which have enriched my hands-on experience in tackling diverse web development challenges. I value writing clean, well-structured, and maintainable code.
